# AGENT Policies

## Purpose

This is the master policy file for AI assistants working in this repository.

It points to the policy documents under [docs/agents](./docs/agents/) and should be treated as the top-level entry point for implementation, review, calibration, testing, and learner-facing output.

## Usage rules

- Prefer the most specific policy for the task.
- If multiple policies apply, follow all of them together.
- When policies overlap, prioritize safety, correctness, calibration integrity, and user-facing clarity.
- Keep project docs, scoring logic, and user-facing behavior aligned.
