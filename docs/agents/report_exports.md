# Report Export Policy

## Goals

- Keep exported reports useful outside the UI.
- Preserve core content across Markdown, Word, and PDF.

## Rules

- Include question, essay, score summary, category details, and improvement guidance.
- Keep section order consistent across formats.
- Prefer readability over heavy styling.
- Ensure exports reflect the same result payload shown in the frontend.

## Limitations

- layout may differ by format
- long text may wrap differently
- style parity with the UI is not required
